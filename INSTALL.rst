Installation
============

Invenio-Base is on PyPI so all you need is:

.. code-block:: console

   $ pip install invenio-base

You can install a development version and run tests using:

.. code-block:: console

    $ cd invenio-base
    $ pip install -e ".[tests]"
    $ ./run-tests.sh
